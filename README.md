# 💰 Gestor de Gastos Personales

Aplicación móvil desarrollada para ayudar a los usuarios a administrar sus finanzas personales de forma simple e intuitiva.

La aplicación permite registrar gastos, organizarlos por categorías, visualizar movimientos recientes y obtener resúmenes que facilitan el control del presupuesto personal.

## 🚀 Funcionalidades

### 🔐 Autenticación de usuarios
- Inicio de sesión.
- Registro de nuevas cuentas.
- Recuperación de contraseña.

### 📊 Dashboard Principal
- Visualización de los últimos gastos registrados.
- Resumen rápido de la actividad financiera.
- Acceso a las principales funcionalidades de la aplicación.

### ➕ Registro de Gastos
- Ingreso de monto.
- Selección de categoría.
- Descripción opcional.
- Almacenamiento de movimientos para análisis posterior.

### 📈 Análisis de Gastos
- Visualización de gastos agrupados por categorías.
- Seguimiento de hábitos de consumo.
- Información clara para la toma de decisiones financieras.

### 📄 Exportación de Reportes
- Generación y exportación de reportes en formato PDF.
- Resumen de gastos para compartir o guardar.

### 👤 Perfil de Usuario
- Consulta de información personal.
- Gestión de datos de la cuenta.

## 🎯 Objetivo

Brindar una herramienta sencilla para que cualquier persona pueda registrar, controlar y analizar sus gastos diarios, obteniendo una visión clara de su situación financiera y fomentando mejores hábitos de ahorro.
